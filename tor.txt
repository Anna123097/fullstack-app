чат минимум
  1.клиент должен сообщить серверу об изменении текстового поля 
    - заметить изменения в текстовом поле 
    - прочитать содержимое текстового поля
    - отправить текст на api endpoint сервера
  2.сервер должен запомнить присланный ему текст 
    - отличить этот запрос от других запросов
    - собрать тело запроса в строку
    - сохранить строку 
  3.сервер должен вписать этот текст в текстовое поле при отправке страницы
    - получить разметку
    - найти текстовое поле 
    - вставить в него текст 
    - отправить изменненую разметку клиенту

  Правила соединения по которым клиент должен отправить запрос и сервер ответить, потом соединение разрывается

  Правила соединения по которым клиент должен устанавливать соединение к серверу,  и общаться можно сколько хочешь, и потом разорвать соединение
